# Contributing

Thanks for your interest in improving *Stunspot's Guide to Game Theory*.

This repository is a curated, model-facing knowledge canon. Useful contributions include typo fixes, clarity improvements, broken-link repair, metadata alignment, citation-quality notes, corpus-hygiene concerns, and source-traceability improvements.

## Repository Policy

Preserve the directory structure:

- `docs/` is navigation and GitHub Pages guidance.
- `knowledge-packs/by-report/` contains canonical individual source reports.
- `knowledge-packs/compiled-packs/` contains grouped upload packs.
- `knowledge-packs/omnibus/` contains the whole-corpus bundle.
- Do not create `docs/reports/`.

## Contribution Notes

- Keep file paths valid and stable.
- Preserve author attribution and license terms.
- Do not remove reserved hero/social image references merely because final images are not committed yet.
- When changing corpus content, identify the affected source report and explain why the change improves source accuracy, clarity, or model/RAG ingestion quality.
- Contributions are provided under the same license unless otherwise agreed in writing.
